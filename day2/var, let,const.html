Difference between var ,let and const in JavaScript

Ans:-

  • var , let and const keywords - is used to declare variable in js.
  • The var variables belong to the global scope or local scope is limited to the functio‚Äù within which it is defined.If it is defined outside any function, the scope of the variable is global. var is function scoped.
  • The let and const variables are blocked scopes.The scope of a variable defined with the keyword let or const is limited to the block defined by curly braces[ {} ].
  • const variables is limited to the block defined by curly braces [ {}]. However if a variable is defined with keyword const, it cannot be reassigned.
