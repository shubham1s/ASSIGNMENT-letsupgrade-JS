Explain with examples the remaining methods of String and Array

String Methods

  • trim() - The trim() method removes whitespace from both sides of a string
  • search() - The search() method returns the position of the first occurrence of a specified text in a string
  • slice() - The slice() method extract a part of a string and returns the extracted parts in a new string
  • substring() - The substring() method extract a part of a string and returns the extracted parts in a new string
  • charAt() - The charAt() method returns the character at a specified index (position) in a string
  • charCodeAt() - The charCodeAt() method returns the unicode of the character at a specified index in a string

Array Methods

  • toString() - The toString() method returns an array as a comma separated string
  • concat() - The concat() method is used to merge (concatenate) arrays
  • slice() - The slice() method slices out a piece of an array into a new array
  • reverse() - The reverse() method reverses the elements in an array

JS Code:- 


        // String Methods
        let s = "      java.prog.com is developed By Shubham     ";
        s = s.trim(); // Removes white-spaces
        console.log(s);

        console.log(s.search('SHUBHAM')); //returns the position of the first occurrence value/index

        console.log(s.slice(28,33));  //extract a part of a string  

        console.log(s.substring(0,7)); // gives a substring and the index 0 to 7

        console.log(s.charAt(28)); //returns the character at index 28

        console.log(s.charCodeAt(28)); // returns the unicode of the character at index 28

        // Array Methods
        let arr = ['Monitor','Keyboard','Mouse','CPU'];
        let arr2 = ['Speakers','Pendrive'];

        console.log(arr.toString()); //returns an array as a comma separated string

        console.log(arr.concat(arr2)); // merge (concatenate) 2 arrays

        console.log(arr.slice(3)); //create new array with a value at index 3
        
        console.log(arr.reverse()); // reverses the elements of an array
